The following API provides a path that allows an IP address or a Domain to be passed along with the base URL. The API will return a JSON format file with all data from "https://whois.whoisxmlapi.com/".

To get started, please provide a search parameter (URL or Domain) following the template below:

https://cyderes-skills-api-yf2nwdd3oq-uw.a.run.app/api/ <IP-OR-DOMAIN>

Sample Request for the domain "google.com":

https://cyderes-skills-api-yf2nwdd3oq-uw.a.run.app/api/google.com
