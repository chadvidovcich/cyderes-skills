Cyderes Skills Assessment

The following API provides a path that allows an IP address or a Domain to be passed along with the base URL. The API will return a JSON format file with all data from Whois API.

To get started, please provide a search parameter (URL or Domain) following the template below:

https://cyderes-api-gw-aa2v0pof.wl.gateway.dev/api/ [IP_OR_DOMAIN]

Sample Request for the domain "google.com":

https://cyderes-api-gw-aa2v0pof.wl.gateway.dev/api/google.com

Sample Request for the IP address "69.63.176.13":

https://cyderes-api-gw-aa2v0pof.wl.gateway.dev/api/69.63.176.13
